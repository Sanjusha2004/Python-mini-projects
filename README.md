## 🚀 Projects Included

### 1. Acronyms Generator

Generates an acronym from a given phrase.

Example
Input: Random Access Memory
Output: RAM

---

### 2. BMI Calculator

Calculates Body Mass Index using height and weight.

Formula used:
BMI = weight / (height²)

---

### 3. Email Slicer

Splits an email address into username and domain.

Example
Input: [example@gmail.com](mailto:example@gmail.com)

Output
Username: example
Domain: gmail.com

---

### 4. Password Generator

Generates a strong random password using letters, numbers, and symbols.

---

### 5. Number Guessing Game

A simple game where the computer generates a random number and the user tries to guess it.

Features

* Random number generation
* Hints (Too High / Too Low)

---

### 6. QR Code Generator

Generates a QR code for a given text or URL.

Example use cases

* Website links
* Messages
* Contact information

---

### 7. Story Generator

Creates fun random stories by combining different words and phrases.
